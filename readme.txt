Requires at least: 4.0
Tested up to: 6.1
Stable tag: 4.2

== Description ==

This plugin will allow you or support staff to check the licence status of customers using their website address.

== Installation ==

Once plugin has been installed, go to Settings >> General to enter in your Freemius API details (Dev ID, Public Key and Secret Key).
Once these are added, you can go to FS License Check page to check the status of a customers licence.
